Prevent comment spam automatically by using Javascript.

== Description ==

When entering a comment the plugin will load some Javascript values - unique for the visitor - to ensure that the user is a real person.

If Javascript is not enabled in the visitor's browser, an error message will be shown when submitting the comment.

The spam protection is not enabled for logged-in users.

== Installation ==

1. Activate the plugin.
1. No configuration neccessary.
